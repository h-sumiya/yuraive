Yuraiveは、音声、画像、動画をつないで分岐のある作品を作成し、AndroidまたはWindowsで再生できるメディアプレイヤーです。

作品を作るときはブラウザ版の[Yuraive Editor](https://editor.yuraive.com/)を使います。
