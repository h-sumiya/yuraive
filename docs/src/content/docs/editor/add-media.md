---
title: 素材を追加する
description: 音声、画像、動画、字幕をドラッグしてMedia Nodeへ割り当てます。
---

エディタは、コンテンツフォルダ内の素材を左側のファイルツリーに表示します。

## 認識するファイル

| 種類 | 拡張子                                                    |
| ---- | --------------------------------------------------------- |
| 音声 | `.mp3`、`.wav`、`.ogg`、`.m4a`、`.aac`、`.flac`           |
| 動画 | `.mp4`、`.webm`、`.mov`、`.m4v`                           |
| 画像 | `.png`、`.jpg`、`.jpeg`、`.webp`、`.gif`、`.avif`、`.svg` |
| 字幕 | `.vtt`                                                    |

再生できる音声と動画のコーデックは端末によって異なるため、配布先のAndroid版とWindows版でも確認してください。

字幕はUTF-8のWebVTTで保存します。

## 一つの素材からノードを作る

1. ファイルツリーで音声または動画を探します。
2. 素材を中央のキャンバスの空いている場所へドラッグします。
3. 作成されたMedia Nodeを選びます。
4. 右側のインスペクターでID、表示名、重み、音量などを確認します。

画像をキャンバスへドラッグした場合は、画像付き音声の候補が作成されます。

その候補には音声も割り当ててください。

## 音声へ画像を付ける

1. 音声を含むMedia Nodeを作ります。
2. ファイルツリーの画像をそのノードへドラッグします。
3. インスペクターでメディアの種類が音声と画像になったことを確認します。
4. 画像の表示方法を`cover`、`contain`、`stretch`から選びます。

画像は、同じノードにある画像未設定の音声へ割り当てられます。

## 複数の音声を一つのノードへ追加する

ファイルツリーの音声または動画を既存のMedia Nodeへ順にドラッグします。

同じノードの候補から、重みに応じて再生時に1件が選ばれます。

フォルダをMedia Nodeへドラッグすると、その中の音声と動画を一括で候補へ追加できます。

フォルダをキャンバスの空いている場所へドラッグすると、フォルダ内の音声と動画をまとめた新しいMedia Nodeが作成されます。

## コンピューターから素材を取り込む

OSのファイル管理画面から音声、動画、またはそれらを含むフォルダをキャンバスへドラッグできます。

ChromeまたはEdgeでは、素材がコンテンツフォルダへコピーされ、Media Nodeへ登録されます。

同名ファイルがある場合は上書きされることがあるため、取り込み前に名前を確認します。

## 字幕を付ける

ファイルツリーの`.vtt`を、字幕を付ける音声または動画のMedia Nodeへドラッグします。

字幕は先頭のメディア候補へ割り当てられるため、複数候補がある場合はインスペクターで対象を確認します。

## 素材が表示されない場合

ファイルツリーの再読み込みボタンを押します。

拡張子が一覧にないファイルは素材として割り当てられないため、対応形式へ変換します。

詳しい対処は[エディタの保存と表示の問題](/troubleshooting/editor/)にあります。
