---
title: ボタンと再生設定を作る
description: 選択ボタン、ボタンレイアウト、再生画面の操作可否をグラフへ接続します。
---

作品内の選択肢は、ノードとは独立した**ボタン**として作り、表示するノードと押した後のノードへ接続します。

## ボタンを作る

1. ツールバーの「ボタン」を押します。
2. 作成されたボタンを選びます。
3. インスペクターでボタンIDと表示文字列を入力します。
4. 必要に応じて表示開始時刻、表示終了時刻、色、文字サイズなどを設定します。

同じボタンを複数ノードへ接続すると、表示内容と押下時の遷移を共有できます。

## ボタンを表示するノードへつなぐ

1. Media Node下部のポートから線をドラッグします。
2. ボタン上部のポートへ接続します。

ボタンは、接続したMedia Nodeを再生している間だけ表示対象になります。

接続していないボタンはプレイヤーに表示されず、エディタでは警告になります。

## 押した後の遷移をつなぐ

1. ボタン右側のポートから線をドラッグします。
2. 遷移先ノード左側のポートへ接続します。
3. 遷移先が複数ある場合は線の重みを設定します。

ボタンを押すと現在のメディアが終了し、ボタンからの遷移だけを使って次のノードが選ばれます。

## ボタンレイアウトをつなぐ

ボタンがあるMedia Nodeには、ボタンを配置する`.yuraive-layout.html`が必要です。

新規グラフでは既定のレイアウトが作成されます。

1. ツールバーの「レイアウト」を押し、使用するファイルを選びます。
2. ツールバーの「再生設定」を押します。
3. レイアウトノード下部から、再生設定上部のポートへ接続します。
4. Media Node上部から、再生設定下部のポートへ接続します。

ファイルツリーのレイアウトを再生設定上部へ直接ドラッグして接続することもできます。

ボタンの「配置先」には、レイアウト内のslot名を入力します。

配置先が存在しないボタンは検査エラーになります。

レイアウト自体を変更する方法は[ボタンレイアウトを作る](/authoring/layouts/)で説明します。

## 再生画面の操作を設定する

再生設定を選ぶと、次の表示と操作を切り替えられます。

- 停止を許可する
- シークバーを表示する
- 再生時間を表示する
- シークを許可する
- シーン名を表示する
- ファイル名を表示する
- 次へを許可する
- 前へを許可する
- アクセント色を指定する

ノードへ個別の再生設定を接続しない場合は、グラフ全体の既定設定を使用できます。

アクセント色は`#RRGGBB`形式で指定し、黒または白に近すぎる色は操作要素が見えにくくなるため検査エラーになります。
