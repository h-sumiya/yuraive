---
title: 再生の流れを作る
description: Media Node、遷移、開始、終端、重みを設定して作品の流れを作ります。
---

Yuraiveの再生順は、素材を持つ**Media Node**とノード間の遷移で作ります。

プレイヤーは開始ノードから再生し、メディアの終了またはボタン操作をきっかけに次のノードへ進みます。

## Media Nodeを作る

ツールバーの「メディアNode」を押すか、キャンバスの空いている場所をダブルクリックします。

素材をキャンバスへドラッグして、素材入りのMedia Nodeを直接作ることもできます。

ノードを選ぶと、右側のインスペクターで次の項目を編集できます。

- ノードID
- 画面に表示する名前と色
- 開始ノードまたは終端ノード
- メディア候補と重み
- 適用する再生設定

ノードIDはグラフ内で重複させず、後から見ても役割が分かる名前にします。

## 開始ノードを決める

最初に再生するノードを選び、インスペクターの「開始ノード」を有効にします。

一つのグラフに開始ノードは必ず1件だけ必要です。

開始ノードには別のノードやボタンから接続できません。

接続済みのノードを開始にすると、入力側の接続を解除する確認が表示されます。

## 終了時の遷移をつなぐ

1. 遷移元ノードの右側にあるポートを押すか、そこから線をドラッグします。
2. 遷移先ノードの左側にあるポートを選びます。
3. 2件以上の遷移先がある場合は、それぞれの線の重みを設定します。

重みは比率であるため、合計を100にする必要はありません。

たとえば重み`1`と`3`なら、前者が4分の1、後者が4分の3の割合で選ばれます。

ツールバーの「重み」と「確率」を切り替えると、線上の表示を確認しやすくなります。

## 終端ノードを作る

最後のノードを選び、インスペクターの「終端ノード」を有効にします。

終端ノードはメディアの再生後に作品を完了し、終了時遷移とボタンを持てません。

終端ノードのメディアをループさせることもできません。

## 線とノードを修正する

線を選ぶと、遷移を削除したり、線の途中へ新しいノードを追加したりできます。

ノードを選んで`Delete`または`Backspace`を押すと、そのノードと接続を削除します。

右クリックメニューの「前後を接続して削除」は、単純な中間ノードを取り除くときに使えます。

接続操作を途中でやめるときは`Esc`を押します。

## 条件によって遷移先を決める

履歴や時刻などの条件で遷移先を決める場合は、ツールバーの「Script Node」を使います。

Script Nodeの作り方は[Starlarkで分岐を作る](/authoring/starlark/)で説明します。
