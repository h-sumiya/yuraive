---
title: Windows版を使う
description: Windows版のライブラリ、再生、Androidへの共有を使い始めます。
---

Windows版では、端末内、SMB、WebDAVの作品を追加し、広い画面に合わせたライブラリとプレイヤーを利用できます。

一般公開は準備中のため、配布開始後の入手先は[Yuraive公式サイト](https://yuraive.com/)で案内します。

64ビット版のWindows 10 version 1809以降とWebView2 Runtimeが必要です。

## 最初のフォルダを追加する

1. ライブラリの「フォルダを追加」を押します。
2. 端末内の作品には「この PC」を選びます。
3. Windowsのフォルダ選択画面で、作品を含むフォルダを選びます。
4. 追加されたフォルダまたは作品をライブラリから開きます。

SMBとWebDAVの手順は[コンテンツを追加する](/library/add-content/)で確認できます。

## 作品を再生する

作品を選ぶと内容が検査され、エラーがなければ再生画面が開きます。

横長の画面ではプレイヤーが2列になり、十分に広い画面ではライブラリとプレイヤーが並びます。

画面上の戻る操作に加えて、マウスの戻るボタンでもプレイヤー、フォルダ、履歴、設定から戻れます。

![Windows版Yuraiveの再生画面](/screenshots/windows-player.png)

## Androidへライブラリを共有する

1. 「設定」を開きます。
2. 「端末間接続」の「共有を有効化する」を有効にします。
3. ライブラリ上部の「Androidと接続」を開いてQRコードを表示します。
4. Android版で「フォルダを追加」、「Windows」の順に押してQRコードを読み取ります。

共有を停止するときは「共有を有効化する」を無効にします。

接続コードを作り直す場合は、QRコード画面の「接続コードを更新」を使います。

## ローカル作品を編集する

ローカルフォルダのメニューから「エディタで開く」を選ぶと、Windows版に内蔵されたYuraive Editorで編集できます。

内蔵エディタは選んだフォルダへ直接保存できます。

SMBとWebDAVのフォルダは内蔵エディタで開けないため、ローカルへコピーしてから編集します。
