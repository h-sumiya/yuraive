Yuraiveでは、プレイヤーとエディタを目的に応じて使い分けます。

| 目的                                 | 使うもの                                     |
| ------------------------------------ | -------------------------------------------- |
| 作品を探して再生する                 | Android版またはWindows版                     |
| 再生順や選択肢を作る                 | Yuraive Editor                               |
| レイアウトや分岐条件を細かく調整する | Yuraive Editorのレイアウト編集とStarlark編集 |

## 対応環境

Android版はAndroid 8.0以降に対応します。

Windows版は64ビット版のWindows 10 version 1809以降に対応します。

Yuraive Editorはブラウザで動作し、ChromeまたはEdgeでは選んだフォルダへ変更を直接保存できます。

ほかの対応ブラウザではフォルダを読み込めますが、保存したファイルはダウンロードされます。

Android版とWindows版の一般公開は準備中です。

公開状況は[Yuraive公式サイト](https://yuraive.com/)で案内します。

## 再生までの流れ

1. 作品のフォルダを端末、SMB、WebDAVのいずれかに用意します。
2. プレイヤーの「フォルダを追加」から作品を含むフォルダを選びます。
3. ライブラリに表示された作品を選んで再生します。

プレイヤーは編集用の`.yuraive.json`と、配布用の`.yuraive`を読み込めます。

同じフォルダに同名の両形式がある場合は、配布用の`.yuraive`だけが表示されます。

## 制作までの流れ

1. 音声、画像、動画を一つのコンテンツフォルダにまとめます。
2. [Yuraive Editor](https://editor.yuraive.com/)でそのフォルダを開きます。
3. 素材をキャンバスへドラッグし、再生の流れを線でつなぎます。
4. 「プレビュー」で再生と選択肢を確認します。
5. 画面下のエラーを解消して保存します。

最初の作品は、[フォルダを開いて保存する](/editor/open-and-save/)から作成できます。
