---
title: Android版を使う
description: Android版のライブラリ、再生、Windows接続を使い始めます。
---

Android版では、端末内、SMB、WebDAV、接続したWindowsのライブラリを同じ画面から利用できます。

一般公開は準備中のため、配布開始後の入手先は[Yuraive公式サイト](https://yuraive.com/)で案内します。

## 最初のフォルダを追加する

1. ライブラリ画面の「フォルダを追加」を押します。
2. 端末内の作品には「端末」を選びます。
3. Androidのフォルダ選択画面で、作品を含むフォルダを選びます。
4. ライブラリへ戻り、追加したフォルダまたは作品を開きます。

選んだフォルダの読み取り権限は、アプリを終了した後も保持されます。

SMBとWebDAVの手順は[コンテンツを追加する](/library/add-content/)で確認できます。

![Android版Yuraiveのライブラリ画面](/screenshots/android-library.png)

## 作品を再生する

作品を押すと内容が検査され、エラーがなければ再生画面が開きます。

警告だけがある場合は内容を確認してから再生できますが、エラーがある作品は再生できません。

再生画面からライブラリへ戻っても再生は続き、画面下のミニプレイヤーから再生画面へ戻れます。

ミニプレイヤーを下へスワイプすると再生を停止します。

Android 13以降では、バックグラウンド再生のために初回再生時の通知許可を求められます。

![Android版Yuraiveの再生画面](/screenshots/android-player.png)

## Windowsのライブラリを追加する

1. Windows版の「設定」で「共有を有効化する」を有効にします。
2. Windows版のヘッダーに表示される「Androidと接続」を開きます。
3. Android版で「フォルダを追加」、「Windows」の順に押します。
4. Android版のカメラでWindows版のQRコードを読み取ります。

接続後はWindowsのフォルダがAndroidのライブラリに表示されます。

メディアはWindowsとAndroidの間で直接送られ、接続の待ち合わせに使うサービスはメディア本体を中継しません。

直接通信できないネットワークでは接続できないため、同じネットワークで試すか[接続の対処法](/troubleshooting/remote-library/)を確認してください。
