---
title: 履歴と再生統計を見る
description: 再生セッション、履歴の書き出し、作者定義の再生統計を確認します。
---

再生履歴には、実際に再生したメディアの開始時刻、終了時刻、実再生時間、終了理由が記録されます。

一時停止中の時間は実再生時間に含まれません。

## 再生履歴を開く

1. ライブラリ上部の履歴ボタンを押します。
2. 作品名、開始日時、実再生時間、完了状態を確認します。
3. 履歴の作品を選ぶと、その作品があるライブラリの場所へ移動します。

履歴は一つの再生開始から終了までをセッションとしてまとめて表示します。

Script Nodeとメディアのない即時遷移は履歴を増やしません。

保持される履歴は最大1,000件です。

## 履歴を書き出す

履歴画面の書き出しボタンから、保存中の履歴をJSON Lines形式のファイルへ書き出せます。

JSON Linesでは、1行が一つの再生記録です。

書き出したファイルには作品のメディア本体や認証情報は含まれません。

## 履歴を削除する

履歴画面の削除ボタンを押し、確認画面で削除を確定します。

この操作は保存中の全履歴を削除し、元に戻せません。

お気に入りとライブラリのフォルダは削除されません。

## 再生統計を見る

作者が再生統計を用意した作品では、再生画面のグラフ形のボタンから「再生統計」を開けます。

表示内容と並び順は作品のStarlarkスクリプトが作成します。

統計を用意していない作品では、このボタンを使用できません。

作者が共有内容を用意した場合は、共有ボタンを押した後にテキストを確認し、必要なら編集してから共有します。

共有ボタンを押すまでは外部アプリを開かず、自動投稿も行いません。
