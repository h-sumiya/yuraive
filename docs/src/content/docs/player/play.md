---
title: 作品を再生する
description: 再生、一時停止、シーク、分岐ボタン、前後移動を使います。
---

作品を選ぶと、プレイヤーはグラフと参照ファイルを検査してから開始ノードを再生します。

## 再生画面の操作

| 操作               | 動作                                               |
| ------------------ | -------------------------------------------------- |
| 再生または一時停止 | 現在の音声または動画を再生、一時停止する           |
| シークバー         | 現在のメディア内を移動する                         |
| 前へ               | 同じ再生セッションの直前の確定履歴へ戻る           |
| 次へ               | 現在のメディアを完了として扱い、終了時の分岐へ進む |
| 作品のボタン       | 作者が設定した遷移先へ進む                         |
| ハート             | 作品をお気に入りへ追加、または削除する             |
| 情報               | 作品名、説明、作者、外部リンクを表示する           |
| 歌詞               | WebVTT字幕がある場合に内容を表示する               |

作品の設定により、シーク、前へ、次へ、停止などが非表示または操作不能になることがあります。

利用者が操作制限を外す必要がある場合は、「設定」の「プレイヤーコントロール」で「すべて表示・許可」を有効にします。

この設定は作品ファイルを書き換えず、作品が指定したアクセント色とボタンレイアウトは維持します。

## 分岐ボタンを使う

作品のボタンは、シーンの開始直後ではなく指定された時刻に表示されることがあります。

ボタンを押すと現在のメディアが終了し、そのボタンに接続された次のノードへ進みます。

複数の遷移先が設定されている場合は、作者が指定した重みに従って遷移先が選ばれます。

## ライブラリへ戻る

再生画面から戻っても、停止操作を行わない限り再生は続きます。

Android版では、再生画面を下へスワイプしてライブラリへ戻り、ミニプレイヤーを下へスワイプすると停止できます。

Windows版では、プレイヤーの戻るボタンまたはマウスの戻るボタンでライブラリへ戻れます。

## 中断後に再開する

プレイヤーは前回の再生状態を保存し、アプリを起動し直した後に復元できます。

ただし、「停止」で終了したセッションは復元情報が削除されるため、続きから再開できません。

最初から再生し直すと、新しい再生セッションとして履歴が記録されます。

## 再生エラーが出た場合

「再試行」を押す前に、ライブラリを更新し、作品の参照ファイルが存在するか確認します。

解決しない場合は[読み込みと再生の問題](/troubleshooting/content/)を確認してください。
