---
title: AIアシスタントで使う
description: ChatGPTのカスタムGPTと、Claude Desktop、Claude Code、Codex向けのYuraive Skillを案内します。
---

AIアシスタントを使うと、Yuraiveの操作方法やコンテンツ制作について質問し、Yuraiveファイルの検査や修正を依頼できます。

| 使うAI         | 利用方法                                                                                                                         |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| ChatGPT        | [カスタムGPT「Yuraive Guide」を開く](https://chatgpt.com/g/g-6a5ba2dd3c548191b114cb312c6548a4-yuraive-guide)                     |
| Claude Desktop | [Yuraive Skill 1.0.0をダウンロード](https://github.com/h-sumiya/yuraive/releases/download/skills-v1.0.0/Yuraive-skill-1.0.0.zip) |
| Claude Code    | [Yuraive Skill 1.0.0をダウンロード](https://github.com/h-sumiya/yuraive/releases/download/skills-v1.0.0/Yuraive-skill-1.0.0.zip) |
| Codex          | [Yuraive Skill 1.0.0をダウンロード](https://github.com/h-sumiya/yuraive/releases/download/skills-v1.0.0/Yuraive-skill-1.0.0.zip) |

SkillのZIPは、各製品のSkill追加手順に従って読み込んでください。

Yuraive Skillは、アプリとエディタの操作案内、Yuraiveファイルの検査と修正、レイアウトやStarlarkの作成を支援します。
