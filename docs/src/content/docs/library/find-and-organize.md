---
title: 作品を探して整理する
description: ライブラリの検索、お気に入り、最近の作品、シャッフルを使います。
---

ライブラリでは、追加したフォルダと見つかった作品を一つの画面から探せます。

## 検索する

1. ライブラリ上部の検索ボタンを押します。
2. フォルダ名、作品名、作者名の一部を入力します。
3. 検索結果からフォルダまたは作品を選びます。

検索は大文字と小文字を区別しません。

タグは作品情報に表示されますが、現在の検索対象には含まれません。

## 作品をお気に入りにする

作品のメニューから「お気に入りに追加」を選ぶか、再生画面のハートを押します。

お気に入りを外すときは、同じ操作をもう一度行います。

ライブラリの「最近のお気に入り」には、最近追加したお気に入りから順に作品が表示されます。

## 目的別の一覧を使う

| 一覧             | 表示する作品                          |
| ---------------- | ------------------------------------- |
| 再生した作品     | 履歴に残っている作品                  |
| 最近追加         | ライブラリで新しい作品                |
| 最近のお気に入り | お気に入りにした作品                  |
| シャッフル       | 再生できる作品から無作為に選ばれた1件 |

シャッフルで作品を選べない場合は、ライブラリを更新し、エラーのない作品があることを確認します。

## 作品のファイルを確認する

作品のメニューから「作品情報とアセット」を開くと、作者、Content ID、日時、タグ、参照ファイルを確認できます。

参照ファイルに「見つかりません」または「不正なパス」と表示された作品は、再生前に作者へ修正を依頼します。

作品情報の表示だけではファイルを変更しません。

## フォルダをライブラリから外す

フォルダの削除操作は、その場所をライブラリから外します。

端末、SMB、WebDAVにある元のフォルダとファイルは削除されません。
